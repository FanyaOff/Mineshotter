# Mineshotter

Mod that helps you make clean screenshots **IN ANY GUI**: select an area, draw on it if needed, then copy it or upload and get a link.

![-01-02-202621_48_22-ezgif com-speed](https://github.com/user-attachments/assets/fde724f0-69b1-4bb9-a763-7eb5dfa0f240)


---

## Features

- Simple screenshot editor (select area + optional drawing)
- Move mode: move/resize the selected area
- Draw mode: draw lines inside the selected area + color palette
- Copy button: copies the final image to clipboard
- Upload button: uploads the final image and copies the link to clipboard
- Optional Mod Menu settings (needs Mod Menu + Cloth Config)

---

## Controls

- **F4** — open the editor (can be changed in **Options → Controls**)
- **Left Click** — select area / move / draw (depends on mode)
- **Enter** — confirm selection
- **M** — switch Move / Draw
- **Ctrl + C** — copy and close
- **Ctrl + Z** — undo last stroke (Draw mode)
- **Esc** — close palette / reset current selection step
